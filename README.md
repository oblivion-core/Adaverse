# ⚜ ADAVERSE

> *Your gateway to academic excellence and professional growth.*

A dark-themed academic library platform built as a frontend web project by students at **Université Akli Mohand Oulhadj – Bouira**. ADAVERSE offers a structured interface for browsing courses, exercises, and academic resources organized by subject.

## ✨ Features

- 🏛️ **Grand library aesthetic** — dark gold theme inspired by classical academic architecture
- 📚 **Course catalog** — subjects organized by semester with resource cards (Analyse, Algèbre, ASD, Physique, and more)
- 🎓 **Certifications & community** sections
- 🔐 **Login & signup pages** — user authentication UI
- 🎵 **Ambient background music** for immersive experience
- 📱 **Responsive design** — works on desktop and mobile

---

## 🛠️ Tech stack

- **HTML5** — structure and content
- **CSS3** — custom dark gold theme, responsive layout
- **JavaScript** — interactivity and UI behavior
- **Vercel** — deployment and hosting
